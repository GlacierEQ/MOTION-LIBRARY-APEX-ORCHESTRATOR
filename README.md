## Quick Start

```bash
# Clone apex orchestrator
git clone https://github.com/GlacierEQ/MOTION-LIBRARY-APEX-ORCHESTRATOR.git
cd MOTION-LIBRARY-APEX-ORCHESTRATOR

# Initialize submodules (all 23 repos)
git submodule update --init --recursive

# Install apex dependencies
npm install
pip install -r requirements.txt

# Configure Hawaii court credentials
cp .env.example .env
# Edit .env with JEFS credentials

# Launch apex control panel
npm run apex:start
```

## Core Commands

```bash
# Generate motion using integrated pipeline
apex motion:generate --type="emergency" --court="hawaii-family"

# Process document through intelligence tier

# File across multiple jurisdictions
apex file:multi --jurisdictions="hawaii-supreme,federal-district"

# Track all deadlines
```
